# Styleguide

The styleguide records UI and UX patterns that agents should reuse instead of inventing new ones.

Document patterns for:

- tables
- forms
- filters
- navigation
- destructive actions
- loading states
- empty states
- errors
- badges
- modals
- sidebars
- reusable components

Current pattern notes:

- [Table filters](./table-filters.md)
- [Table bulk edit](./table-bulk-edit.md)

Add or update styleguide docs when a change introduces or changes a reusable UI pattern.
