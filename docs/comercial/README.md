Material de trabajo para el contacto empresarial con constructoras y empresas con alta carga documental. Pensado para ejecutarse con llamadas telefónicas, email, WhatsApp y reuniones en vivo.

## Regla de oro

Cada contacto termina **siempre** con un próximo paso agendado con fecha concreta (otra llamada, una reunión, un envío). Un contacto sin próximo paso es un prospecto perdido.

## El relato central

> "Digitalizamos y estructuramos la documentación de empresas constructoras usando IA, para reducir carga manual, errores y falta de trazabilidad."

Todo guion, mensaje y reunión orbita alrededor de esa frase.
